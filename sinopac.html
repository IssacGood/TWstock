⚠️ 說明

永豐 Shioaji API 需要 CA 憑證（本地檔案），無法在 GitHub Actions 上使用。
請在本機電腦安裝 pip install shioaji 後，用下方產生的腳本在本地執行，
執行後會產生 data/sinopac_quotes.json，再手動 git push 上傳即可。

🔐 API 金鑰僅儲存於本地瀏覽器（localStorage），不會上傳到 GitHub。

API 設定（本地儲存）
產生本機執行腳本

點擊「產生腳本」後，複製下方程式碼，儲存為 fetch_sinopac.py 在你的專案根目錄，再執行。 


      

    

  


  
  

    
使用步驟

    

      
1
安裝永豐套件：pip install shioaji

      
2
在上方填入 API 設定後，點「產生腳本」

      
3
複製腳本內容，儲存為 fetch_sinopac.py 放在專案根目錄

      
4
在終端機執行：python fetch_sinopac.py
會產生 data/sinopac_quotes.json

      
5
上傳到 GitHub：git add data/sinopac_quotes.json && git commit -m "update sinopac" && git push

      
6
可以設定 Windows 工作排程器 或 Mac/Linux 的 cron 自動每天執行

    


    

      💡 模擬帳號說明：

      永豐提供模擬環境（simulation=True），可以用自己的帳號免費測試，

      但行情資料為模擬報價，不完全等同真實市場。

      如需真實行情，需使用正式帳號 + CA 憑證。
